# GTFS Editor

A browser-based tool for creating custom GTFS transit maps. Add stops, draw routes, and export a valid GTFS ZIP for use in [MapCalipers](https://github.com/joshspicer/MapCalipers).

**Live:** [joshspicer.com/gtfs-editor](https://joshspicer.com/gtfs-editor)

## Usage

1. Click the map to add stops
2. Use the route tool to draw routes between waypoints
3. Export as a GTFS ZIP — import directly into MapCalipers via the share sheet or file picker

No server required — everything runs in the browser with localStorage persistence.
